# Tanks-but-its-online
Online Python tank game with a max of 8 players for now 

_______________________________________________________________________________________
# ABOUT
_______________________________________________________________________________________

- This is a 2D Online Tank Game, with the sole purpose of converting my Local multiplayer version into an online one, and making it better. And for fun

_______________________________________________________________________________________
# HOW TO PLAY
_______________________________________________________________________________________

- Move: WASD
- Shoot: Left click

_______________________________________________________________________________________
# FEATURES
_______________________________________________________________________________________

- 2D Physics with pymunk
- Powerups (Damage, Heal, Speed etc...)
- 8 Different coloured tanks
- Minimap, showing vague locations of players and powerups (intentional dont worry :D)
- Max of 8 players at once... (Could change in the future, well see)

_______________________________________________________________________________________
# INSTALLATION
_______________________________________________________________________________________

1- Download the "Tanks.exe", along with the assets folder.
2- Run the exe!
